# ISO/IEC 27001 Annex A Controls (Demo)

| Control_ID | Control_Name                        | Control_Category      | Description                                             | Implementation_Guidance |
|-----------|-------------------------------------|-----------------------|---------------------------------------------------------|------------------------|
| A.5.1     | Information security policies       | Organizational        | Management direction and support for information security aligned with business requirements. | Publish, approve, and review policies annually. |
| A.5.7     | Threat intelligence                 | Organizational        | Collect and analyze threat information to anticipate incidents. | Define sources, integrate feeds into monitoring, brief stakeholders. |
| A.6.2     | Information security awareness      | People                | Ensure personnel are aware of their security responsibilities. | Run onboarding + annual training, track completion. |
| A.6.7     | Remote working                      | People                | Secure remote work arrangements. | Provide secure VPN, enforce endpoint hardening, define acceptable use. |
| A.7.4     | Physical security monitoring        | Physical              | Detect and respond to physical security events. | Use CCTV/access logs, retain evidence logs, review alerts. |
| A.8.2     | Access control                      | Technological         | Limit access to authorized users and devices. | Implement RBAC, MFA, periodic access reviews. |
| A.8.12    | Data leakage prevention             | Technological         | Prevent unauthorized disclosure of information. | Use DLP tooling, classify data, monitor e-mail & cloud channels. |
| A.8.16    | Secure development lifecycle        | Technological         | Embed security into software development. | Apply secure coding standards, threat modeling, code reviews. |
