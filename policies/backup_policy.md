# Backup Policy (Demo)

## Purpose
Ensure critical data is backed up and recoverable.

## Scope
All production databases, file shares, and SaaS exports.

## Policy Statement
Critical systems are backed up daily with encrypted storage; non-critical weekly. Restoration tests occur quarterly.

## Roles & Responsibilities
- Infrastructure Team: maintain backup schedules and logs.
- Security Team: review encryption keys.

## Compliance
Backup logs and restore tests are reviewed during audits.
